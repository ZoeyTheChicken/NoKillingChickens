Have you been looking for a plugin that stops Players from killing Chickens on your server? If you have, you have found the perfect plugin! Simply named NoKillingChickensV2 or NKC-v2.
This plugin supports Bukkit, Spigot, Parrot, Paper servers that run 1.19.4.

There will be other compiles for 1.20, 1.18, 1.17, 1.16, 1.15, 1.14 and 1.13.

You can download this plugin and apply it to your server!
